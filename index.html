Safe Review Pipeline

Local NLP pipeline to moderate user reviews: BERT-based toxicity detection, summarization, rewriting — no cloud, no compromise on privacy.

Learn more

This project is a fully local, explainable moderation pipeline that: scrapes reviews, detects harmful content using BERT, generates smart summaries with T5, and rewrites toxic comments for publication — all with no external API. It includes a Streamlit dashboard and a FastAPI backend, and is designed to be modular, privacy-preserving, and production-ready.

Hybrid Music Recommender

Intelligent playlist recommender mixing Word2Vec & NLP embeddings. Interactive app, filters by genre & duration, exportable results.

Learn more

This hybrid music recommender blends collaborative filtering (Word2Vec) and content-based embeddings (SentenceTransformers). It includes a Streamlit interface with search, sliders to adjust semantic weights, and genre/duration filters. The system is 100% local, customizable, and ready for large datasets. Visual explanations and CSV exports are also available.

ReviewGuardian – Safe Review Pipeline

A fully local AI system to moderate user reviews in real-time. It detects toxic content, explains decisions, and rewrites harmful text, all while preserving user privacy.

Learn more

ReviewGuardian is a privacy-focused NLP system for identifying and managing toxic content in user reviews. It includes:

  • End-to-end review pipeline: scraping, preprocessing, and custom text cleaning
  • Multi-model toxicity detection using Naive Bayes, Logistic Regression, and Random Forest with TF-IDF
  • Explainability layer powered by SHAP for local interpretation of predictions
  • Model versioning and evaluation with built-in A/B testing and performance tracking
  • Fully local deployment: Streamlit front-end and FastAPI backend, no cloud, full control

All inference runs locally to guarantee complete data privacy and maximum transparency.

Technologies used

Python TF-IDF Naive Bayes Logistic Regression Random Forest SHAP FastAPI Streamlit scikit-learn

Hybrid Music Recommender

Intelligent playlist recommender combining Word2Vec and content-based NLP embeddings. Fully local, interactive and privacy-focused: explore music by mood, genre, era and more.

Learn more

This premium music recommender system blends collaborative filtering and semantic content analysis:

  • Dual similarity engine combining Word2Vec (co-occurrence) and Sentence-BERT (semantic) embeddings
  • Interactive Streamlit interface with real-time filters: genre, duration, decade, mood
  • Audio-based exploration mode using features like energy, danceability, and valence
  • Visual dashboard with interpretability tools: heatmaps, radar charts, and performance benchmarking

  • 100% offline system: no cloud, no API, full data privacy

Built with performance, explainability, and personalization in mind, a powerful sandbox for recommender systems.
